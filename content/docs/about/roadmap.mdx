## Current Status

OpenArcades is in its **v1 foundation** phase. The core platform infrastructure is in place, including the game hosting system, documentation, and contributor workflow.

## Near Term

- **First community games** — accepting game contributions from the community
- **Game thumbnails** — automatic screenshot generation for games without thumbnails
- **Search improvements** — full-text search across games and documentation
- **Mobile experience** — optimized touch controls and responsive game frames

## Medium Term

- **Leaderboards** — optional high-score tracking per game
- **Game collections** — curated themed lists (e.g., "Retro Classics", "Brain Teasers")
- **Categories and tags** — enhanced filtering and discovery
- **Analytics** — play counts and popularity ranking

## Long Term

- **Multiplayer support** — real-time multiplayer via WebRTC or WebSocket
- **Game API** — standardized JavaScript API for games to communicate with the platform
- **User accounts** — save progress and favorites
- **Game jams** — time-limited game creation events

## Contributing to the Roadmap

Have an idea? Open an issue on GitHub to suggest features or improvements. Pull requests are welcome for any items on this roadmap.
